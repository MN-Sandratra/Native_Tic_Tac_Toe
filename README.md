# Tic Tac Toe Game

A modern and stylish Tic Tac Toe game built with React Native and Expo.

## Installation

1. Clone the repository

2. Install dependencies
```bash
cd tic-tac-toe
npm install
```

3. Start the development server
```bash
npm start
```
